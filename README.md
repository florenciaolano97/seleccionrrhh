# ALBA v2 — Módulo 5: entrevistas estructuradas

## Funciones incorporadas

- Plantillas predeterminadas:
  - entrevista inicial general;
  - competencias STAR;
  - jóvenes profesionales y pasantías;
  - liderazgo.
- Creación de plantillas personalizadas.
- Banco de preguntas editable.
- Competencia, indicadores, orden, obligatoriedad y puntaje por pregunta.
- Respuestas por texto, audio o ambas.
- Asignación de entrevistas a una postulación.
- Fecha límite y mensaje para el candidato.
- Portal del candidato para comenzar y completar la entrevista.
- Almacenamiento y reproducción de respuestas de audio.
- Asistencia automática transparente para respuestas escritas.
- Detección de indicadores, extensión y componentes STAR.
- Fortalezas y aspectos para profundizar por pregunta.
- Evaluación humana con puntaje y evidencia.
- Decisión final humana con justificación obligatoria.
- Seguimiento de estados:
  - invitada;
  - en curso;
  - completada;
  - evaluada;
  - vencida;
  - cancelada.
- Permisos específicos para entrevistas.
- Auditoría de asignaciones y evaluaciones.

## Límites de la asistencia automática

No se evalúan voz, acento, apariencia, gestos ni emociones.
El audio se almacena para revisión humana. La asistencia automática
solo trabaja con la respuesta escrita o la transcripción incorporada.

## Instalación

Reemplazar en GitHub:

- app.py
- requirements.txt
- runtime.txt
- README.md

Hacer commit y reiniciar Streamlit Cloud. Verificar que la barra lateral
muestre `Versión 5.0.0`.
