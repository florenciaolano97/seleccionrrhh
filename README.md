# ALBA v2 — Módulo 4: candidatos, CV y ranking

## Funciones incorporadas

- Perfil profesional editable del candidato.
- Carga de CV en PDF y DOCX.
- Extracción de nombre, correo, teléfono y secciones profesionales.
- Carga masiva de hasta 50 CV por búsqueda.
- Creación o actualización automática de candidatos.
- Ranking por coincidencia con criterios del puesto.
- Desglose de coincidencias y brechas.
- Recomendaciones AVANZA, REVISAR o BAJA COINCIDENCIA.
- Pipeline con etapas configurables.
- Descarga del CV.
- Recalculo manual del puntaje.
- Auditoría de cargas y cambios de etapa.

## Protección contra sesgos

El puntaje no utiliza:

- nombre;
- correo;
- DNI;
- edad o fecha de nacimiento;
- teléfono;
- ciudad.

El resultado es una ayuda de preselección y no reemplaza la decisión humana.
