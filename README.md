# ALBA v2 — Módulo 4.1

## Funciones nuevas

- Banco de candidatos por empresa.
- Asignación individual o masiva a búsquedas.
- Asignación directa a una etapa del pipeline.
- Filtros por país, provincia, localidad y edad.
- El filtro de edad es administrativo y está excluido del scoring.
- Criterios predeterminados según el tipo de búsqueda.
- Plantillas para Tecnología, Ingeniería, Compras, RR. HH.,
  Comercial y Administración/Finanzas.
- Edición de términos, pesos, obligatoriedad y umbrales.
- Recalculo automático de todas las postulaciones.
- Razones detalladas de cada recomendación.
- Fortalezas, brechas y puntos por dimensión.
- Decisión manual del reclutador con razón obligatoria.
- Notas internas.
- Favoritos.
- Etiquetas.
- Exportación del pipeline filtrado a CSV.
- Compatibilidad con las bases de los módulos anteriores.

## Instalación

Reemplazar en GitHub:

- app.py
- requirements.txt
- runtime.txt
- README.md

Hacer commit y reiniciar Streamlit Cloud. Verificar que aparezca
`Versión 4.1.0` en la barra lateral.
