# Alba — Módulo 1 autocontenido

Esta versión no depende de carpetas `modules` ni de imports internos.

## Archivos que deben estar en la raíz del repositorio

```text
app.py
requirements.txt
runtime.txt
README.md
```

## Streamlit Cloud

1. Eliminá del repositorio el `app.py` anterior.
2. Subí estos cuatro archivos a la raíz.
3. Confirmá que el archivo se llame exactamente `app.py`.
4. En Streamlit Cloud seleccioná `app.py`.
5. Reiniciá la aplicación.

## Ejecutar localmente

```bash
pip install -r requirements.txt
streamlit run app.py
```
